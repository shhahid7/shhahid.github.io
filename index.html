Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://shhahid.github.io/ (Give it the name "shhahid Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "shhahid Repository", and install repository.shhahid-2.0.0.zip.
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "shhahid Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
